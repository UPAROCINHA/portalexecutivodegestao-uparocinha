Sobre o projeto

O Portal Executivo de Gestão da UPA Rocinha é uma plataforma institucional criada para apresentação, acompanhamento e análise dos indicadores administrativos e assistenciais da unidade.

O projeto nasce a partir do Accountability 2026, evoluindo de uma apresentação estática para um ambiente digital interativo.

---

Objetivos

- Centralizar informações estratégicas da unidade.
- Apresentar indicadores assistenciais.
- Facilitar apresentações institucionais.
- Permitir atualização através de planilhas e APIs.
- Criar histórico da evolução da gestão.
